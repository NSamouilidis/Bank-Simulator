Bank Simulator

Welcome to FreeBank!
This is the first version (V1) of the Bank Simulator.

A simple and interactive console-based Java program that lets users simulate basic banking operations. With this program, you can:

- Check your account balance

- Deposit money

- Withdraw funds

- Exit the application

The program is designed to be user friendly, guiding you through each transaction and validating your input to prevent mistakes. It also allows multiple transactions in a single session, giving a realistic banking experience.

Future Versions:

V2: Add a credit card feature with an income-based eligibility system.

V3: Introduce loan options, stock investment features, and a calculator to estimate how many years it would take to become a millionaire based on monthly investments.
